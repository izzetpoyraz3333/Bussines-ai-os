# Business OS - AI Powered SaaS Dashboard 🚀

A modern, highly responsive, and feature-rich business management dashboard built with Next.js and Tailwind CSS. The core highlight of this project is the integration of a custom AI assistant powered by Google Gemini and n8n, designed to help users navigate business data seamlessly.

## ✨ Key Features
* **Intelligent AI Assistant:** Integrated via n8n webhooks and Google Gemini API to provide real-time, context-aware responses.
* **Fully Responsive Design:** Carefully crafted with Tailwind CSS to ensure a flawless experience across all devices (Desktop, Tablet, Mobile).
* **Modern UI/UX:** Features a sleek dark/light mode, smooth slide-in mobile menus, and an intuitive layout.
* **Serverless Architecture:** Deployed on Vercel for lightning-fast performance and seamless CI/CD.

> **⚠️ Important Note on AI Assistant:** Due to the current underlying system and webhook configurations, the AI assistant generates responses without using Turkish characters (e.g., using 'c' instead of 'ç', 's' instead of 'ş').

## 🛠️ Tech Stack
* **Frontend:** Next.js, React, Tailwind CSS
* **Backend/Integration:** n8n (Workflow Automation), Google Gemini AI
* **Deployment:** Vercel
